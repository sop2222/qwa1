Имитатор лампочек складской навигации для dev окружений.

Подключается к указанному в конфиг.файле MQTT брокеру и ловит сообщения с
адресом ячейки, анализирует указанный в сообщении оттенок и длительность.
Визуализирует полученную команду в соём ГУЕ как кружок в заданных
координатах поверх картинки.

пример картинки Resources\inactiveDeck.jpg
пример конфиг. файла Resources\qwa1.json
пример сообщения в указанной в конфиг.файле теме в MQTT
в teez.atlassian.net/wiki/spaces/TD/pages/334823428/dev
на всякий случай дублирую:
{ "colorR": "100",
  "colorG": "100",
  "colorB": "200",
  "illuminateState": 1,
  "cellID": "A.1.L.22.6.6.3",
  "illuminateTime": "100",
  "fallBackState": 0 }
